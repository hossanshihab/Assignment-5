Important FAQ

What is the Document Object Model (DOM)?

The DOM is a dynamic representation of the document meaning it can be modified by JavaScriptcode. The DOM is not the same as the HTML source code of the document. Instead its a parsed representation of the document that can be accessed and manipulated by programming languages like JavaScript.

How do you select an element from the DOM?

There are some many ways to select an element from the DOM const element = document.getElementById('element');

What is event delegation in the context of the DOM, and why is it useful?

When an event occurs on a child element the event bubbles up the DOM tree triggering the event listener on the parent element. The parent element can then determine which child element triggered the event and respond accordingly.

How do you manipulate an element's attributes and styles using the DOM?

const element = document.getElementById('element'); const style = element.style.cssText; console.log(style); element.style.cssText = 'background-color: red; color: white;'
